A modern JavaFX desktop application for managing personal diary entries.

## Features
- **Rich Text Editor**: Format diary entries with styling options
- **Entry Management**: Create, read, update, delete entries
- **Advanced Search**: Real-time search with filters
- **Theme Support**: Light/dark mode toggle
- **Statistics & Analytics**: Entry insights and export functionality
- **Auto-save**: Automatic saving of entries

## Technologies
- JavaFX 17+ (with Web module for HTML editor)
- File I/O for persistence
- Java 11+

## How to Run
1. Clone repository
2. Open in NetBeans
3. Run `DiaryManagerGUI.java`
