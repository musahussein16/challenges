# Aero Dynamics Weather Widget ✈️

A professional aviation weather dashboard for pilots and flight planners.

## Features
- **Flight Planning Panel**: Select departure/arrival airports
- **Critical Flight Indicators**: Visibility, crosswind, turbulence, precipitation
- **Real-time Updates**: Dynamic flight status and conditions
- **Safety-Focused Design**: Color-coded alerts for aviation professionals
- **Professional Interface**: Clean, technical layout with precise data display

## Technologies
- JavaFX 17+
- Maven for dependency management
- Java 11+

## How to Run
1. Ensure Java 11+ and JavaFX 17 are installed
2. Clone the repository
3. Open in NetBeans as JavaFX Maven project
4. Run `AeroDynamicsWeatherWidget.java`
